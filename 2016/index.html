Utilizando Tornado e Python3 para construção de Microservices

Microservices esta cada vez mais entrando em cena, e nada mais justo que usar Python para desenvolver. Na palestra será abordado o uso do python3 no desenvolvimento de serviços, com: Tornado, SqlAlchemy ( ORM ) e Celery ( agendamento de tarefas e tarefas assíncronas ).

Joelmir Ribacki

Desenvolvedor Python na empresa DevGrid UK, estudante de Ciência da Computação na UCS, cofundador da startup AoMercado. Experiência no desenvolvimento de aplicativos híbridos ( Phonegap ), construção e manutenção de sistemas ( Django ) e desenvolvimento de sistemas com microservices ( Tornado* ).

Aprendendo Python para se tornar um Sysadmin Melhor

Para ser um bom sysadmin é necessário adquirir o máximo de recursos técnicos disponíveis. Pretendo apresentar os motivos que me fizeram ir atrás dessa linguagem, compartilhar algumas das experiências que tive com Python e mostrar como esse conhecimento adquirido pode auxiliar na área de administração de sistemas.

Carlos Alberto Parisotto

Programação Assíncrona em Python

É possível programar de forma assíncrona em Python e existem algumas bibliotecas para nos ajudar nisso, como o Python é green thread programar de forma assíncrona pode aumentar a performance. O Python 3.5 vem com syntax sugar para facilitar a programar de forma assíncrona sem a necessidade de usar bibliotecas.

Estratégias de Testes de Software

O objetivo desta palestra é explorar estratégias de testes de software e como elas se relacionam pra otimizar tempo de execução, qualidade e feedback rápido. Falaremos de pirâmides de testes, integração e entrega contínua, tipos de testes(unitário, integração, funcional, mutação), técnicas(TDD, BDD, etc) e como isso impacta na qualidade e na manutenção de um projeto de software . Exemplos em Python :)

Giovane Liberato

Giovane Liberato é consultor desenvolvedor na ThoughtWork onde trabalha com práticas ágeis e de desenvolvimento extremo. Atualmente interessado em desenvolvimento seguro de aplicações, qualidade de software e arquiteturas distribuídas.

O paradigma Orientado a Objetos no Python

Python é uma linguagem de programação de alto nível "livre" que suporta diversos paradigmas, como o orientado a objetos, imperativo, funcional e procedural. A palestra tem como objetivo introduzir o participante no paradigma Orientado a Objetos, explicando conceitos e sintaxes, além de exemplificar de forma prática o paradigma na linguagem.

Admirador do software livre, foi moderador do Viva o Linux durante quase 7 anos, entre 2009 e 2015. Geração de Fármacos com Python

Existe uma área de pesquisa da Química Orgânica de grande interesse atualmente, a obtenção de fármacos com o auxílio da computação científica. A possibilidade de projetar novos compostos com propriedades bem definidas evita os altos custos da síntese experimental exploratória em alta escala. Dessa forma, foi desenvolvido pela Unicamp uma nova fórmula de modelagem molecular, baseada na metodologia QSAR-4D, chamada de LQTAGrid, que ajuda na criação de novos remédios. A ideia da plaestra é apresentar a experiência de migrar o núcleo de processamento matemático do sistema para Python e criar uma interface Web em Django para abstrair os diversos processos necessários para modelagem e análise molecular.
